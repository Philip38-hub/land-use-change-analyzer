How It Works

  1. Create a new project
  2. Upload two aerial images from different years
  3. Classify land use in each image
  4. Analyze and visualize changes between the two images

The system uses K-means clustering to identify different land use types (urban, vegetation, water, etc.) and compares them to detect changes over time.
